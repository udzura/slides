# Fukuoka.edge とは

- エッジランタイム、サーバーレス、Wasm などを扱う勉強会
- Cloudflare Workers に限らず、Fastly Compute、Deno Deploy、Vercel Edge なども歓迎
- 実装の話、運用の話、失敗談、LT、なんでも持ち寄る

# なぜ今エッジなのか

- ユーザーに近い場所でコードを動かせる
- 小さな API や認証、キャッシュ制御を軽く置ける
- プラットフォームの機能が増えて、実アプリにも使いやすくなってきた
